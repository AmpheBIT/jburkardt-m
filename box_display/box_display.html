BOX_DISPLAY
Integer Data Box Plots

BOX_DISPLAY is a MATLAB program which creates a plot over a range of integer boxes, whose default color is gray, but some of which can be painted red (new stuff) and some painted blue (old stuff)

Usage:

box_display ( m, n, @blue, @red, title )
where
  • m and n are the number of rows and columns of boxes (actually, the range will include one extra box in each direction, labeled "0");
  • @blue a function handle, or an anonymous function, returning a logical value. If blue(x,y) is true, the box will be colored blue.
  • @red a function handle, or an anonymous function, returning a logical value. If red(x,y) is true, the box will be colored red, unless it was already blue. Thus, the red boxes indicate "new" information.
  • title is a title for the plot.
