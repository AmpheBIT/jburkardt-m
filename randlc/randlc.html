RANDLC
NAS Parallel Benchmark Pseudorandom Number Generator

RANDLC is a MATLAB library which implements a version of the pseudorandom number generator used by the NAS Parallel Benchmarks.

The generator has the form 

        X(K+1) = A * X(K) mod 2^46
where the suggested valule of the multiplier A is 5^13 = 1220703125.

This scheme generates 2^44 numbers before repeating.

The web site for the NAS Parallel Benchmarks is http://www.nas.nasa.gov/Resources/Software/npb.html.

Source Code:

  • randlc.m, generates the next pseudorandom value.
  • randlc_jump.m, jumps ahead to the K-th pseudorandom value.

Examples and Tests:

  • randlc_test.m, calls all the tests
  • randlc_test01.m, just calls RANDLC ten times.
  • randlc_test02.m, calls RANDLC ten times and examines the SEED.
  • randlc_test03.m, demonstrates how the SEED can be changed to give different results.
  • randlc_test04.m, demonstrates how RANDLC_JUMP can jump ahead in the pseudorandom sequence.
  • randlc_test_output.txt, the output file.
