PPMA_IO
Read or write an ASCII PPM file.

PPMA_IO is a MATLAB library which can read or write an ASCII PPM (Portable Pixel Map) file, involving 3 NROW by NCOL integer arrays of RGB color information.

There are many graphics programs, such as XV and GIMP, that can display images in this formats, or translate them to other formats such as BMP, GIF, JPEG, MacPaint, X Window, and so on.
