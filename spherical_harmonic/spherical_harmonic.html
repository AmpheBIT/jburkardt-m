SPHERICAL_HARMONIC
Evaluate Spherical Harmonic Functions

SPHERICAL_HARMONIC is a MATLAB library which evaluates spherical harmonic functions.

Source Code:

  • legendre_associated_normalized.m, evaluates the associated Legendre functions P(N,M)(X), normalized for use in spherical harmonic calculations;
  • legendre_associated_normalized_values.m returns values of the normalized associated Legendre functions.
  • r8_factorial.m, computes the (real) factorial function;
  • spherical_harmonic.m, evaluates the spherical harmonic function;
  • spherical_harmonic_values.m, returns some values of the spherical harmonic function;
  • timestamp.m, prints the current YMDHMS date as a timestamp;

Examples and Tests:

  • spherical_harmonic_test.m, runs all the tests;
  • spherical_harmonic_test_output.txt, is the output from the tests;
  • spherical_harmonic_test01.m, tests LEGENDRE_ASSOCIATED_NORMALIZED.
  • spherical_harmonic_test02.m, tests SPHERICAL_HARMONIC and SPHERICAL_HARMONIC_VALUES.
