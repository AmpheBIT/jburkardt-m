QUAD_TASKS
Quadrature using Tasks

QUAD_TASKS is a MATLAB program which demonstrates how a numerical quadrature procedure can be carried out as a set of independent tasks whose results are combined at the end.

The task computing feature of MATLAB's Parallel Computing Toolbox is used to manage the creation, execution, and analysis of the multiple tasks.

Source Code:

  • quad_job_local.m, sets up a job as a collection of tasks and runs it locally.
  • quad_local_output.txt, the output from a local run of the tasks.
  • quad_job_ithaca.m, sets up a job as a collection of tasks and runs it on the Ithaca cluster.
  • quad_ithaca_output.txt, the output from a run of the tasks on the Ithaca cluster.
  • quad_fsu.m, sets up a job as a collection of tasks and runs it on the FSU HPC cluster.
  • quad_fsu_output.txt, the output from a run of quad_fsu on the FSU HPC cluster.
