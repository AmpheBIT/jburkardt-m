UNIFORM_DATASET
Generate Uniform Random Datasets

UNIFORM_DATASET is a MATLAB program which creates a uniform random dataset and writes it to a file.

Usage:

r = uniform_dataset ( m, n, seed )
where
  • m is the spatial dimension;
  • n is the number of points to generate;
  • seed is the initial seed value.
  • r is the computed m by n dataset.
The data is written to the file uniform_m_n.txt.
