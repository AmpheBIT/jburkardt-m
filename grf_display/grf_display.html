GRF_DISPLAY
Display a GRF Graph in the MATLAB Graphics Window

GRF_DISPLAY is a MATLAB program which reads a GRF file containing information about an abstract graph and displays a plot in the MATLAB graphics window.

Usage:

grf_display prefix
or
grf_display ( 'prefix' )
where
  • 'prefix'.grf is the name of the GRF file to be read and displayed.
