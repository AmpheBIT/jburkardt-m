PLY_DISPLAY
Display a 3D Graphics PLY File

PLY_DISPLAY is a MATLAB program which reads a 3D graphics PLY file, and displays an image on the screen.

Usage:

ply_display ( 'file.ply' )
where
  • 'file.ply' is the name of the file
