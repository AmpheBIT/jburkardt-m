SHALLOW_WATER_1D_DISPLAY
Display a Solution of the Shallow Water Equations in 1D

SHALLOW_WATER_1D_DISPLAY is a MATLAB program which can display a solution of the shallow water equations in 1D, generally as computed by the program shallow_water_1d.

Related Data and Programs:

SHALLOW_WATER_1D, a MATLAB program which solves the 1D shallow water equations.

Source Code:

  • shallow_water_1d_display.m, a program to display each timestep of the solution; the user hits RETURN to advance.
  • timestamp.m prints the YMDHMS date as a timestamp.

Examples and Tests:

  • sw1d_test.m calls all the tests.
  • sw1d_test01.m displays solutions from the TEST01 data.

The command 

        [ h_array, uh_array, x, t ] = shallow_water_1d ( 41, 100, 1.0, 0.2, 9.8 )
created the files:
  • test01_x.txt, the X values.
  • test01_t.txt, the T values.
  • test01_h.txt, the H values.
  • test01_uh.txt, the UH values.
