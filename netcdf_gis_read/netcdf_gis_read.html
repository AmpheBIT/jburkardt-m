NETCDF_GIS_READ
Read NETCDF Files of GLIMMER GIS Data

NETCDF_GIS_READ is a MATLAB program which reads data from a GLIMMER data file, written in the NETCDF format, containing GIS data.

The GIS data includes dimensions:

  • level
  • time
  • x0
  • x1
  • y0
  • y1

The GIS data includes data arrays:

  • acab(x1,y1,time)
  • artm(x1,y1,time)
  • beta(x0,y0,time)
  • bheatflx(x1,y1,time)
  • temp(x1,y1,level,time)
  • thk(x1,y1,time)
  • time(time)
  • topg(x1,y1,time)
  • x0(x0)
  • x1(x1)
  • y0(y0)
  • y1(y1)

Source Code:

  • gis_data_print.m prints the GIS data (usually much too much information!)
  • gis_header_print.m prints the GIS header information.
  • gis_read.m reads a specified GLIMMER GIS file.

Examples and Tests:

  • gis_test.m reads a GLIMMER GIS file and prints the header information.

GIS_5KM.180211 contains data for Greenland at a 5KM resolution.

  • gis_5km.180211.nc, the data.
  • gis_5km.180211_header.txt, the header information as retrieved by NCDUMP.
  • gis_5km.180211_output.txt, the header information as retrieved and printed by running gis_test on the data.
