ASA245
Logarithm of Gamma Function

ASA245 is a MATLAB library which evaluates the logarithm of the gamma function.

ASA245 is Applied Statistics Algorithm 245. Source code for many Applied Statistics Algorithms is available through STATLIB.

Languages:

Related Data and Programs:

TEST_VALUES, a MATLAB library which contains sample values for a number of distributions.

TOMS291, a MATLAB library which evaluates the logarithm of the Gamma function.

Source Code:

  • alngam.m computes the logarithm of the gamma function.
  • gamma_log_values.m returns some values of the Log Gamma function.
  • lngamma.m computes Log(Gamma(X)) using a Lanczos approximation.
  • timestamp.m prints out the current YMDHMS date as a timestamp.
