STOKES_FLOW/TEST_PROBLEMS
TOMS866 / IFISS Test Problems for the Stokes Equations

STOKES_FLOW/TEST_PROBLEMS is a directory of MATLAB M-files which define test problems for the Stokes equations.

Source Code:

  • S1_batch.m data file for test problem S1, channel domain.
  • S2_batch.m data file for test problem S2, backward-facing step.
  • S3_batch.m data file for test problem S3, lid driven cavity.
  • S4_batch.m data file for test problem S4, colliding flow.
  • backwardstep_bc.m backward facing step boundary condition.
  • backwardstep_flow.m backward facing step inflow condition.
  • collide_bc.m colliding flow boundary condition.
  • collide_flow.m colliding inflow condition.
  • forwardstep_bc.m forward facing step boundary condition.
  • forwardstep_flow.m forward facing step inflow condition.
  • leakycavity_flow.m Reference Example 5.3 inflow condition.
  • no_flow.m specifies zero inflow condition.
  • plate_bc.m Reference Example 7.4 boundary condition.
  • plate_flow.m Reference Example 7.4 inflow condition.
  • poiseuille_bc.m Reference Example 5.1 boundary condition.
  • poiseuille_flow.m Reference Example 5.1 inflow condition.
  • quad_bc.m channel flow boundary condition for 4:1 quadrilateral domain.
  • quad_flow.m channel flow inflow condition for 4:1 quadrilateral domain.
  • regcavity_flow.m default inflow condition for Reference Example 5.3.
  • stokes_testproblem.m sets up Reference Examples 5.1 to 5.4.
  • tightcavity_flow.m alternative inflow condition for Reference Example 5.3.
  • zero_bc.m sets boundary conditions for Reference Examples 5.3 and 5.4.
