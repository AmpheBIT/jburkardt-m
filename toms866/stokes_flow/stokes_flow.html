STOKES_FLOW
TOMS866 / IFISS Files for the Stokes Equations

STOKES_FLOW is a directory of MATLAB M-files which are specifically for the solution of the Stokes equations.

Source Code:

  • flowbc.m imposes inflow boundary condition.
  • forwardstep_stokes.m set up flow problem in forward step domain.
  • helpme_stokes.m Stokes flow problem interactive help.
  • infsup.m computes inf-sup eigenvalue distribution.
  • localbc_xy.m imposes vector BC for Poisson error estimator.
  • longstep_stokes.m set up flow problem in extended step domain.
  • pipe_stokes.m set up inflow/outflow problem in square domain.
  • plate_stokes.m set up flow problem in slit domain.
  • q1div.m computes norm of divergence of Q1 flow solution.
  • q2div.m computes norm of divergence of Q2 flow solution.
  • quad_stokes.m set up Stokes problem in quadrilateral domain.
  • solve_step_stokes.m solve Stokes problem in step domain.
  • solve_stokes.m solve Stokes problem in square domain.
  • specific_flow.m Reference problem 5.3 default inflow condition.
  • square_stokes.m set up flow problem in unit square domain.
  • step_stokes.m set up flow problem in standard step domain.
  • stokes_q1p0.m vectorized Q1-P0 matrix generator.
  • stokes_q1q1.m vectorized Q1-Q1 matrix generator.
  • stokes_q2p1.m vectorized Q2-P1 matrix generator.
  • stokes_q2q1.m vectorized Q2-Q1 matrix generator.
  • stokespost_q1p0_bc.m postprocesses Poisson error estimator.
  • stokespost_q1p0_p.m computes Poisson error estimator for Q1-P0.
  • stokespost_q1q1_bc.m postprocesses Poisson error estimator.
  • stokespost_q1q1_p.m computes Poisson error estimator for Q1-Q1.
  • stressjmps_q1p0.m stress jumps for rectangular Q1-P0 grid.
  • stressjmps_q1q1.m stress jumps for rectangular Q1-Q1 grid.

Subdirectories:

  • test_problems, example test problems for the Stokes codes.
