DATAFILES
TOMS866 / IFISS Data Files for the Convection-Diffusion Equation

DATAFILES is a directory which is simply a convenient location for IFISS programs to store or retrieve datafiles that hold information about the solution of a specific problem. These files are typically in MATLAB's ".mat" format. Thus, the information in a file named "data.mat" can be retrieved by the MATLAB command "load data".
