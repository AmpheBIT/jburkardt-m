DIFFUSION/TEST_PROBLEMS
TOMS866 / IFISS Test Problems for the Diffusion Equation

DIFFUSION/TEST_PROBLEMS is a directory of MATLAB M-files which define test problems for the diffusion equation, also known as the Poisson equation.
