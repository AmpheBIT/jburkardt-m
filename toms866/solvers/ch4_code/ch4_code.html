SOLVERS/CH4_CODE
TOMS866 / IFISS Solvers
Codes from Chapter 4

SOLVERS/CH4_CODE is a directory of MATLAB M-files which can be used to reproduce results from Chapter 4 of the Elman-Silvester-Wathen monograph.

Source Code:

  • cd_gmres_test.m test preconditioned GMRES for convection-diffusion equation.
  • gs_bottom_to_top.m test bottom-to-top Gauss-Seidel iteration.
  • gs_four.m test four-directional Gauss-Seidel iteration.
  • gs_rb.m test line red-black Gauss-Seidel iteration.
  • gs_top_to_bottom.m test top_to_bottom Gauss-Seidel iteration.
  • helpme_ch4.m helpme for Chapter 4 specialized functions.
  • hor_to_ver.m reorder matrix from horizontal to vertical.
  • m_gs_four.m four-directional Gauss-Seidel preconditioning.
  • m_gs_one.m one-directional Gauss-Seidel preconditioning.
  • m_gs_rb.m line red-black Gauss-Seidel preconditioning.
  • m_ilut_2dir.m two-directional ILU preconditioning.
