SOLVERS
TOMS866 / IFISS M Files for Solving Problems

SOLVERS is a directory of M files used to compute the solution of the various problems.

Source Code:

  • a_cdt.m matrix-vector product for convection-diffusion operator.
  • a_nst.m matrix-vector product for linearized Navier-Stokes operator.
  • bicgstab_ell.m bicgstab(ell) for right-preconditioned iterates.
  • bicgstab_ell_r.m bicgstab(ell) iteration with right preconditioning.
  • cg_test.m CG convergence demo.
  • givapp.m apply a sequence of Givens rotations.
  • gmres_r.m GMRES iteration with right preconditioning.
  • helpme_it.m iterative solvers interactive help.
  • helpme_mg.m geometric multigrid interactive help.
  • ilu0.m incomplete factorization with no fill of sparse matrix.
  • it_solve.m driver for iterative solution of predefined problem.
  • m_amgt.m algebraic multigrid preconditioner for convection-diffusion operator.
  • m_bfbt.m ideal least squares commutator preconditioner (unscaled).
  • m_diagt.m action of diagonal preconditioning operator.
  • m_fp.m ideal pressure convection-diffusion preconditioner.
  • m_fp_amg.m AMG iterated pressure convection-diffusion preconditioner.
  • m_fp_mg.m GMG iterated pressure convection-diffusion preconditioner.
  • m_ilut.m incomplete LU preconditioner.
  • m_mg.m GMG preconditioner for scalar problems.
  • m_nonet.m "no preconditioning" operator.
  • m_st_block.m block preconditioner for Stokes equations.
  • m_st_mg.m block MG preconditioner for Stokes equations.
  • m_sxbfbt.m ideal stabilized least squares commutator preconditioner.
  • m_xbfbt.m ideal least squares commutator preconditioner.
  • m_xbfbt_amg.m AMG iterated least squares commutator preconditioner.
  • m_xbfbt_mg.m GMG iterated least squares commutator preconditioner.
  • mg_apsetup_q1.m Q1 pressure diffusion matrix for GMG.
  • mg_cavity_domain.m square cavity Q2 grid generator for GMG.
  • mg_cd.m preconditioner for convection-diffusion problem.
  • mg_cd_setup.m GMG convection-diffusion problem on square domain.
  • mg_diff.m GMG preconditioner for diffusion problem.
  • mg_diff_q2q1grid.m Q2-Q1 element grid generator for GMG.
  • mg_diff_setup.m GMG diffusion problem on square domain.
  • mg_diff_setup_ell.m GMG diffusion problem on L-shaped domain.
  • mg_diff_setup_step.m GMG diffusion problem on step domain.
  • mg_ellblock.m prolongation for part of L-shaped and step domains.
  • mg_iter.m performs one GMG iteration.
  • mg_ns.m GMG preconditioner for Navier-Stokes equations.
  • mg_ns_cd.m GMG for convection-diffusion problem (Navier-Stokes).
  • mg_ns_cd_setup.m GMG convection-diffusion problem (Navier-Stokes).
  • mg_ns_diff.m GMG for diffusion problem (Navier-Stokes).
  • mg_ns_diff_lstsq.m GMG for diffusion problem (Navier-Stokes, least-squares).
  • mg_ns_iter.m performs one MG iteration (Navier-Stokes).
  • mg_ns_prolong_step.m GMG prolongation operator for step domain (Navier-Stokes).
  • mg_ns_q1cd.m convection-diffusion matrix generator for GMG (Navier-Stokes).
  • mg_ns_q1cd_supg.m streamline diffusion matrix generator for GMG (Navier-Stokes).
  • mg_ns_smooth.m smoothers for GMG iteration (Navier-Stokes).
  • mg_post.m postsmoothing for GMG.
  • mg_pre.m presmoothing for GMG.
  • mg_prolong.m GMG prolongation operator for square domain.
  • mg_prolong_ell.m GMG prolongation operator for L-shaped domain.
  • mg_prolong_step.m GMG prolongation operator for step domain.
  • mg_q1cd.m convection-diffusion matrix generator for GMG.
  • mg_q1cd_supg.m streamline diffusion matrix generator for GMG.
  • mg_q1diff.m bilinear diffusion matrix generator for GMG.
  • mg_q1grid.m bilinear element grid generator for GMG.
  • mg_smooth.m smoothers for GMG on square domain.
  • mg_smooth_ell.m smoothers for GMG on L-shaped domain.
  • mg_smooth_step.m smoothers for GMG on step domain.
  • mg_solve.m driver for GMG solution of predefined problem.
  • mg_step_domain.m Q2 grid generator for GMG on step domain.
  • mg_zerobc.m imposes zero boundary conditions.
  • resplot.m plot residuals computed by iterative solvers.
