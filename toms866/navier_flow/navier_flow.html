NAVIER_FLOW
TOMS866 / IFISS Files for the Navier-Stokes Equations

NAVIER_FLOW is a directory of MATLAB M-files which are specifically for the solution of the Navier-Stokes equations.

Source Code:

  • fpsetup_q0.m Q0 pressure convection-diffusion matrix.
  • fpsetup_q1.m Q1 pressure convection-diffusion matrix.
  • fpsetup_q2p1.m P1 pressure convection-diffusion matrix.
  • helpme_navier.m Navier-Stokes flow problem interactive help.
  • localbc_xycd.m imposes vector BC for Poisson error estimator.
  • navier_q1.m Q1 convection matrix.
  • navier_q2.m Q2 convection matrix.
  • navierpost_q1p0_bc.m postprocesses Poisson error estimator.
  • navierpost_q1p0_p.m computes Poisson error estimator for Q1-P0.
  • newton_q1.m Q1 convection derivative matrices.
  • newton_q2.m Q2 convection derivative matrices.
  • newtonbc.m imposes Dirichlet bc on Jacobian.
  • null_pressure_index.m index associated with the pressure nullspace.
  • pressurebc.m fixes singularity in Laplacian matrix.
  • solve_navier.m solve Navier-Stokes problem in square domain.
  • solve_plate_navier.m solve Navier-Stokes problem in slit domain.
  • solve_step_navier.m solve Navier-Stokes problem in step domain.
  • Cpre_q1p0.m generate stabilizations for least squares commutator for Q1-P0.
  • Cpre_q1q1.m generate stabilizations for least squares commutator for Q1-Q1.

Subdirectories:

  • test_problems, example test problems for the Navier-Stokes codes.
