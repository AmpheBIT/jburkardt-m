CC_DISPLAY
Display 2D Quadrature Rules

CC_DISPLAY is a MATLAB program which displays the abscissas used in a 2D quadrature rule.

The current version only considers a single Clenshaw Curtis grid, in which the orders of the rule in the X and Y directions may differ.

Eventually, some other variations will be added, in which the points associated with an isotropic Smolyak rule will be displayed, or in which several sets of Clenshaw Curtis grids will be superimposed, or a Gauss-Legendre rule will be displayed.

Source Code:

  • cc_abscissa.m returns the I-th Clenshaw Curtis abscissa.
  • cc_grid.m returns a multidimensional Clenshaw-Curtis grid.
  • cc_grid_display.m displays a multidimensional Clenshaw-Curtis grid.
  • cc_grids_animate.m displays a sequence of merged CC grids from order [1,1] to [A,B].
  • cc_grids_constrained.m computes CC orders and grids satisfying a constraint.
  • cc_grids_constrained_display.m displays grids generated by CC_GRIDS_CONSTRAINED.
  • cc_grids_constrained_size.m computes CC orders and grids with a constraint.
  • cc_grids_minmax.m computes CC orders and grids with Q_MIN <= Q <= Q_MAX.
  • cc_grids_minmax_display.m displays grids generated by CC_GRIDS_MINMAX.
  • cc_grids_minmax_size.m returns sizes for the CC_GRIDS_MINMAX.
  • cc_level_compose_animate.m displays the individual CC grids that compose a given level.
  • cc_level_to_order.m converts CC nesting levels to CC orders.
  • cc_levels_constrained.m computes CC levels and grids satisfying a constraint.
  • cc_levels_constrained_display.m displays grids generated by CC_LEVELS_CONSTRAINED.
  • cc_levels_constrained_animate.m displays a sequence of grids generated by CC_LEVELS_CONSTRAINED.
  • cc_levels_constrained_size.m computes sizes for CC_LEVELS_CONSTRAINED.
  • cc_levels_minmax_animate.m displays a sequence of merged grids generated by CC_LEVELS_MINMAX.
  • cc_levels_minmax_display.m displays grids generated by CC_LEVELS_MINMAX.
  • cc_levels_minmax.m computes CC grids with LEVEL_MIN <= LEVEL <= LEVEL_MAX.
  • cc_levels_minmax_size.m returns sizes for CC_LEVELS_MINMAX.
  • comp_next.m computes the compositions of the integer N into K parts.
  • compnz_next.m computes the compositions of the integer N into K nonzero parts.
  • timestamp.m prints the current YMDHMS date as a time stamp.
  • vec_gray_next.m computes the elements of a product space.
  • vector_constrained_next4.m returns the "next" constrained vector.
