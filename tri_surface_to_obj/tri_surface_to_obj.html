TRI_SURFACE_TO_OBJ
Convert 3d graphics data from TRI_SURFACE to OBJ format

TRI_SURFACE_TO_OBJ is a MATLAB program which reads data from a TRI_SURFACE file describing a triangular mesh of a 3D surface, and converts it to an OBJ file.

A TRI_SURFACE or triangulated mesh surface, is described by a pair of arrays:

  • a node coordinate array containing the coordinates of nodes;
  • a triangle node array containing triples of the indices of the nodes used to form each triangle;

An OBJ file is format for storing a description of the surface of a 3D object, composed of triangles or higher degree polygons.

Usage:

tri_surface_to_obj ( 'node_file', 'element_file', 'obj_file' )
where
  • 'node_file' lists the coordinates of nodes;
  • 'element_file' lists the indices of nodes forming elements;
  • 'obj_file' is the name of the OBJ file to be created.

Examples and Tests:

CUBE is a cube using 8 nodes and 12 triangular faces.

  • cube_nodes.txt the TRI_SURFACE nodes file.
  • cube_triangles.txt the TRI_SURFACE triangles file.
  • cube.obj the OBJ file.
  • cube.png, a PNG image of the shape;

SPHERE is a sphere, using 66 nodes and 119 triangular faces.

  • sphere_nodes.txt the TRI_SURFACE nodes file.
  • sphere_triangles.txt the TRI_SURFACE triangles file.
  • sphere.obj, the OBJ file;
  • sphere.png, a PNG image of the shape;
