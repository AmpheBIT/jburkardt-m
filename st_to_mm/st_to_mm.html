ST_TO_MM
Convert Sparse Matrix File from ST to MM Format

ST_TO_MM is a MATLAB program which reads sparse matrix data from an ST (Sparse Triplet) file and writes it to an MM (Matrix Market) file.

Usage:

st_to_mm ( 'file.st', 'file.mm' )
where
  • 'file.st' is the Sparse Triplet file to be read;
  • 'file.mm' is the Matrix Market file to be created.
