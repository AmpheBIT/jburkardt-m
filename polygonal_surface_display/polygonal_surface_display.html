POLYGONAL_SURFACE_DISPLAY
Display a Polygonal Surface in MATLAB

POLYGONAL_SURFACE_DISPLAY is a MATLAB program which gets the names of the node and face files defining a polygonal surface, reads the information from the files, and displays the surface on the screen.

Usage:

polygonal_surface_display ( 'node_file', 'face_file' )
where
  • 'node_file' is the name of the file containing node information;
  • 'face_file' is the name of the file containing face information.

  • sphere_q4.png, a PNG image of the shape as displayed by POLYGONAL_SURFACE_DISPLAY.
