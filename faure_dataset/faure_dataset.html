FAURE_DATASET
Generate Datasets
of Faure Quasirandom Sequences

FAURE_DATASET is a MATLAB program which creates a Faure quasirandom sequence and writes it to a file.

Usage:

r = faure_dataset ( m, n, skip )
where
  • m is the spatial dimension. This implicitly determines base, which is the smallest prime number greater than or equal to m;
  • n is the number of points to generate;
  • skip is the initial number of points to skip over. The recommended value of skip is base^4-1.
  • r is the m by n array of Faure data.
The data is written to the file faure_m_n.txt.
