NCC_TETRAHEDERON
Newton-Cotes Closed Quadrature for the Tetrahedron

NCC_TETRAHEDERON is a MATLAB library which defines the weights and abscisass for a sequence of 7 Newton-Cotes closed quadrature rules for the tetrahedron.

Newton-Cotes rules have the characteristic that the abscissas are equally spaced. For a tetrahedron, this refers to spacing in the unit reference tetrahedron, or in the barycentric coordinate system. These rules may be mapped to an arbitrary tetrahedron, and will still be valid.

The rules are said to be "closed" when they include points on the boundary of the tetrahedron.

The use of equally spaced abscissas may be important for your application. That may how your data was collected, for instance. On the other hand, the use of equally spaced abscissas carries a few costs. In particular, for a given degree of polynomial accuracy, there will be rules that achieve this accuracy, but use fewer abscissas than Newton-Cotes. Moreover, the Newton-Cotes approach almost always results in negative weights for some abscissas. This is generally an undesirable feature, particularly when higher order quadrature rules are being used.

(Note that the first rule included in the set is not, strictly speaking, a Newton-Cotes closed rule; it's just the rule that uses a single point at the centroid. However, by including this rule as the first in the set, we have a rule with each polynomial degree of exactness from 0 to 6.)

Related Data and Programs:

Source Code:

  • ncc_tetrahedron_degree.m returns the degree of a given NCC rule for the tetrahedron.
  • ncc_tetrahedron_order_num.m returns the order of a given NCC rule for the tetrahedron.
  • ncc_tetrahedron_rule.m returns the points and weights of an NCC rule.
  • ncc_tetrahedron_rule_num.m returns the number of NCC rules available.
  • ncc_tetrahedron_suborder.m returns the suborders for an NCC rule.
  • ncc_tetrahedron_suborder_num.m returns the number of suborders for an NCC rule.
  • ncc_tetrahedron_subrule.m returns a compressed NCC rule.
  • r8mat_det_4d.m returns the determinant of a 4x4 matrix.
  • reference_to_physical_t4.m maps T4 reference points to physical points.
  • tetrahedron_volume.m computes the volume of a tetrahedron.
  • timestamp.m prints the current YMDHMS date as a time stamp.

Examples and Tests:

  • ncc_tetrahedron_test.m, runs all the tests.
  • ncc_tetrahedron_test01.m, tests NCC_TETRAHEDRON_RULE_NUM, NCC_TETRAHEDRON_DEGREE, and NCC_TETRAHEDRON_ORDER_NUM.
  • ncc_tetrahedron_test02.m, tests NCC_TETRAHEDRON_RULE by summing the weights.
  • ncc_tetrahedron_test03.m, tests NCC_TETRAHEDRON_RULE by summing the barycentric coordinates.
  • ncc_tetrahedron_test04.m, tests NCC_TETRAHEDRON_ORDER by integrating monomials in the unit tetrahedron.
  • ncc_tetrahedron_test05.m, tests REFERENCE_TO_PHYSICAL_T4 by transforming an NCC rule from the unit tetrahedron to another tetrahedron.
  • ncc_tetrahedron_test06.m, tests NCC_TETRAHEDRON_RULE by printing out one rule.
  • ncc_tetrahedron_test_output.txt, the output file.
