GFD2
Functions for Geophysical Fluid Dynamics

GFD2 is a MATLAB library which contains many functions for geophysical fluid dynamics.

The MATLAB functions are intended to illustrate the ideas in the textbook Introduction to Geophysical Fluid Dynamics by Benoit Cushman-Roisin and Jean-Marie Beckers.

Related Data and Programs:

GFD2, a dataset directory which contains geophysical data, stored in NetCDF format.

Source Code:

The source codes are listed by the chapter of the book in which they are discussed. A few codes are mentioned in several chapters, and so are listed here several times. The codes are either scripts or functions.

A script is a sequence of MATLAB commands that can be run by invoking the name of the script. Thus, if you are running MATLAB, you can type coriolisdis and know that something will happen.

A function is a sequence of MATLAB commands that require some input values from the user, and return some output values. Thus, to use the function, you need to know what goes in and comes out, and supply the appropriate input. To use aliasanim, for instance, you might type 

        n = 300;
        nper = 2;
        aliasanim ( n, nper )
(This function just draws a picture, so there's no output.)
