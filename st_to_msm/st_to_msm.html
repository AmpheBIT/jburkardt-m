ST_TO_MSM
Convert Sparse Triplet (ST) Matrix File to MATLAB Sparse Matrix

ST_TO_MSM is a MATLAB program which reads sparse matrix data from a Sparse Triplet (ST) file and creates a corresponding MATLAB Sparse Matrix.

Usage:

a = st_to_msm ( 'file.st' )
where
  • 'file.st' is the name of the Sparse Triplet (ST) file to be read.
  • a is the name of the MATLAB Sparse Matrix created.
