ST_TO_MSM
Convert ST Sparse Triplet Matrix File to MATLAB Sparse Matrix

ST_TO_MSM is a MATLAB program which reads sparse matrix data from an ST (Sparse Triplet) file and creates a corresponding MATLAB Sparse Matrix.

Usage:

a = st_to_msm ( 'file.st' )
where
  • 'file.st' is the name of the ST file to be read.
  • a is the name of the MATLAB Sparse Matrix created.
