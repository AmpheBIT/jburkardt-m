PGMA_IO
Read or write an ASCII PGM file.

PGMA_IO is a MATLAB library which reads and writes files in the ASCII PGM (Portable Gray Map) format for 2D grayscale graphics information.

Languages:

PGMA_IO is available in a C version and a C++ version and a FORTRAN77 version and a FORTRAN90 version and a MATLAB version.

Related Data and Programs:

PBMA_IO, a MATLAB library which handles the ASCII Portable Bit Map format.

PGMA, a data directory which contains examples of ASCII PGM files.

PGMA_TO_PGMB, a C++ program which converts an ASCII PGM file to a binary PGM file.

PGMB_TO_PGMA, a C++ program which converts a binary PGM file to an ASCII PGM file.

PPMA_IO, a MATLAB library which handles the ASCII Portable Pixel Map format.
