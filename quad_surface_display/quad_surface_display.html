QUAD_SURFACE_DISPLAY
Display a 3D Surface Defined by a Quadrilateral Mesh

QUAD_SURFACE_DISPLAY is a MATLAB program which can display a 3D surface that is defined by a mesh of quadrilaterals.

The input from the user includes:

  • NODE_NUM, the number of nodes;
  • NODE_XY(1:2,1:NODE_NUM), a list of the coordinates of nodes;
  • QUAD_NODE(1:4,1:QUAD_NUM), a list of the nodes that make up each quadrilateral in the mesh;
  • VALUE_XY(1:NODE_NUM), the value associated with each node;

Usage:

quad_surface_display ( node_num, node_xy, quad_num, quad_node, value )
where
  • node_num, the number of nodes.
  • node_xy, the coordinates of the nodes.
  • quad_num, the number of quadrilaterals.
  • quad_node, the nodes of each quadrilateral.
  • value, the value assigned to each node.
